# プロダクト概要

## VetVoice — 獣医音声診療記録システム

大動物（牛）を診療する日本の獣医師向け音声入力診療記録PoC。
牛舎・農場でスマートフォンから音声入力し、構造化データ→SOAPサマリー→家畜共済記録ドラフトを自動生成する。

## 設計原則

「JSONを正にして、SOAP/共済は派生」
- `transcript_raw` と `extracted_json` が Source of Truth（削除不可）
- `soap_text` / `kyosai_text` は派生出力（再生成可能）

## 対象ユーザー

日本の大動物（牛）診療獣医師。牛舎で手がふさがった状態での利用を想定（モバイルファースト）。

## コンペティション制約 (AWS 10000 AIdeas)

- 予算: AWS無料利用枠 + 最大$200クレジット
- 必須ツール: Kiro
- 推奨サービス: Amazon Bedrock, Amazon Nova
- 審査基準: 革新性、効率性、実用的インパクト

## ドメイン用語

| 用語 | 意味 |
|------|------|
| Visit | 特定の牛に対する1回の診療イベント |
| Cow | QRコード（cow_id）で識別される牛 |
| Extracted_JSON | 音声テキストから抽出された構造化データ（vital, s, o, a, p） |
| Confidence | マスタ照合時の一致度スコア（0.0〜1.0） |
| Unconfirmed | Confidence閾値未満で手動確認が必要な状態 |
| SOAP | Subjective, Objective, Assessment, Plan の診療記録形式 |
| 家畜共済 | 日本の家畜保険制度。診療記録から共済申請書類を生成 |
