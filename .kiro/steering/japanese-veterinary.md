# 日本獣医学ドメイン知識

## 対象動物

本システムは大動物（牛）の診療に特化。小動物（犬猫）や馬の用語体系とは異なる点に注意。

## SOAP形式（獣医診療記録）

獣医診療でのSOAPは人医と同じ4セクション構成だが、内容は畜産特有:

| セクション | 意味 | 獣医での典型例 |
|-----------|------|--------------|
| S (Subjective) | 稟告（飼い主からの聞き取り） | 「昨日から食欲がない」「乳量が減った」 |
| O (Objective) | 所見（獣医師の診察結果） | 体温39.5℃、第一胃蠕動音減弱、脱水あり |
| A (Assessment) | 評価（診断名・病名候補） | 第四胃変位、ケトーシス疑い |
| P (Plan) | 計画（処置・投薬・経過観察） | 右方変位整復術、ブドウ糖静注500ml |

## 病名マスタ（byoumei.csv）

### 構造

CSV形式: `大分類,中分類,小分類,備考`

- 大分類: `コード　名称` 形式（例: `01　循環器病`）— 全角スペース区切り
- 中分類: `コード 名称` 形式（例: `01 心のう炎`）— 半角スペース区切り
- 小分類: `コード 名称` 形式（任意、空の場合あり）
- 備考: 別名・補足（例: `（心膜炎）`）

### マスタコード体系

`大分類コード-中分類コード-小分類コード` 形式

例:
- `01-04-01` → 循環器病 > 心筋症 > 肥大型
- `04-30` → 消化器病 > 第四胃変位（小分類なし）

## 診療点数マスタ（shinryo_tensu_master_flat.csv）

### 構造

CSV形式: `section_id,section_title,item_no,item_name,points_B,points_A,note_no,note_text,aliases,tags,custom_notes`

- section_id: セクションID（例: `S01`）
- item_no: 項目番号（数値）
- points_B: B種点数（共済組合負担分）
- points_A: A種点数（農家負担分）
- note_no/note_text: 注記（同一item_noに複数行の注記がある場合あり）

### マスタコード体系

`section_id-item_no` 形式（例: `S01-1` = 初診、`S01-2` = 再診）

### 主要セクション

| section_id | 内容 |
|-----------|------|
| S01 | 第１診察料（初診・再診・往診） |
| S03 | 第３文書料 |
| S04 | 第４検査料 |
| S05〜 | 処置・手術・投薬等 |

### 往診料の特殊ルール

- 往診距離は片道計算、直線距離基準
- 夜間（18:00-翌8:00）・深夜（22:00-翌5:00）加算あり
- 積雪地域（12/1〜3/31）は追加加算あり → `shinryo_betsu_snow_regions.csv` 参照
- 悪天候時（暴風・暴風雪）加算あり

## 積雪地域マスタ（shinryo_betsu_snow_regions.csv）

### 構造

CSV形式: `prefecture,district,municipality`

- 積雪期間: 12月1日〜翌年3月31日
- 対象: 宮城県、福島県、栃木県、群馬県、山梨県、長野県等の指定市町村
- 一部市町村は旧町村単位で指定（例: `仙台市（旧宮城町、旧秋保町のみ）`）

## 家畜共済制度

### 概要

家畜共済は日本の農業保険法に基づく家畜保険制度。獣医師の診療記録から共済申請書類を作成する。

### 共済記録の必須フィールド

| フィールド | 説明 | データソース |
|-----------|------|------------|
| 牛ID | 個体識別番号 | QRコード（cow_id） |
| 診療日 | 診療実施日 | Visit.datetime |
| 病名 | 診断名（マスタコード付き） | ExtractedJSON.a + byoumei.csv |
| 処置内容 | 実施した処置 | ExtractedJSON.p (type: "procedure") |
| 使用薬剤 | 投与した薬剤と用量 | ExtractedJSON.p (type: "drug") |
| 診療点数 | B種・A種点数 | shinryo_tensu_master_flat.csv |

### 点数の種類

- B種点数: 共済組合が負担する部分
- A種点数: 農家が負担する部分
- 1点 = 金額は組合ごとに異なる（通常1点=数十円）

### Unconfirmed候補の扱い

共済記録では正確なマスタコードが必須。Confidence閾値未満の候補は:
- 該当フィールドを空欄にする
- 「手動入力が必要です」の注記を付与
- 獣医師が手動で正しいコードを選択・確定する

## テンプレートタイプと診療パターン

### 一般診療SOAP (general_soap)

最も汎用的。内科疾患、感染症、中毒等に使用。

### 繁殖SOAP (reproduction_soap)

繁殖関連の診療に特化。キーワード: 妊娠、分娩、繁殖、発情、授精、妊娠鑑定、子宮

### 蹄病SOAP (hoof_soap)

蹄の疾患に特化。キーワード: 蹄、跛行、蹄病、削蹄、蹄底、趾皮膚炎

### 家畜共済テンプレート (kyosai)

共済申請用。マスタコード付きの病名・処置が必須。
